# InventoryApp

This is a very basic inventory tracking application with FIFO cost tracking. It does not have security enabled and is not optimized for mobile viewing.

This is intended as an example of how I approach the same idea with different UI frameworks, in this case MVC, Angular and Blazor.
There is also a React app with less functionality but a more mobile-centered UI.

One known issue is that the footer does not move down when more content is added to the page in the Angular and Blazor apps.
I may fix that someday, but don't feel a lot of pressure to do so.
You can get the idea of how it is meant to work.

Of course, if this was intended to be production-ready, I would set up security, optimize layout for mobile devices and fix that footer issue.
But this is never intended to go into production.
